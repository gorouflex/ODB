# Installation

1. Download [requirements.txt](ODB/requirements.txt) then run 
```
pip install -r [requirements.txt path]
```
or (in macOS)
```
pip3 install -r [requirements.txt path]
```
## In Windows
 1. Click to py file
## In macOS
 1. Open Terminal
 2. Run
```
python3 [ODB.py path]
```
Make sure you have Python installed and a good internet connection!
